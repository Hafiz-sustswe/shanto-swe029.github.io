# Sum Of The Digits Of A Number

***

## Problem Statement

```
    You will be given an integer x. x <= 10^(19). You have to find
    the sum of the digits of the integer and print it.
```

## Example

    Sample Input
        12345
    Sample Output
        15

<br>

    Sample Input
        918273645
    Sample Output
        45
